한국공학대학교 모바일 프로그래밍 프로젝트 – 시흥시머니

📌 프로젝트 소개

시흥시 재정 데이터를 기반으로
예산과 부채 흐름을 직관적으로 시각화하고,
사용자의 참여를 통해 재정에 대한 관심을 높이는 모바일 애플리케이션입니다.

단순 정보 제공을 넘어서,
사용자가 직접 참여하고 의견을 반영할 수 있는 구조를 지향합니다.

🎯 주요 목표
재정 데이터를 직관적으로 이해할 수 있도록 시각화
사용자 참여(투표, 제안)를 통한 재정 관심도 향상
참여 활동에 대한 보상 시스템(포인트 → 지역화폐 교환) 제공
🧩 주요 기능
1. 홈 화면
예산 / 부채 카드 UI
예산 감소, 부채 증가 흐름을 애니메이션으로 표현
분야별 예산 비율 그래프 제공
재정 관련 뉴스 제공
2. 분석 기능
예산 데이터를 상세하게 확인
기간별, 항목별 데이터 분석
3. 참여 기능
재정 관련 설문 및 투표 참여
사용자 참여 활동에 따른 포인트 지급
4. 제안 기능
사용자 의견 작성 및 공유
재정 관련 아이디어 제안
⭐ 차별점
단순 데이터 조회가 아닌 사용자 참여 기반 서비스
참여 활동에 대해 포인트 보상 제공
포인트를 지역화폐로 교환 가능하도록 설계
재정 데이터를 시각적으로 쉽게 이해 가능하도록 구성
